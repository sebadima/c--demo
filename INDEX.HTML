Welcome to PHPTriad for Windows v2.11.

PHPTriad is the basic installer. Additional pieces may be available at this time. Check PHPGeek.com for more details and to ensure that you have installed the most recent version of PHPTriad.

While PHPTriad is provided free and will always remain so, there are costs involved including time and effort as well as some financial costs associated with maintaining this software.   • Fixed Extension path

  • Other misc fixes.

What's new in version 2.1?

  • An all new installer. I've switched to a new software package for building the installer. It's more robust and should make the installer more painless
  • PHP 4.0.5 - the newest version of PHP.
  • Perl is now included. Though I provide no real support for it, it was a highly requested feature and has been added
  • Bug fixes.
  • Includes the following versions:
    • PHP - 4.0.5
    • MySQL - 3.23.32
    • Apache - 1.3.14
    • PHPMyAdmin - 2.1.0
    • Perl - nsPerl 5.005_03
If you wish to add additional extensions, place them in c:\apache\php\extensions. You can enable the extensions that are disabled by default by editing c:\apache\php\php.ini. The shebang line for Perl is #!c:/apache/perl/nsperl.exe.

To see the PHP settings for your installation, open the phpinfo.php file.
